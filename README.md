# Wayfinder - A Pathfinder 1E Character Builder Desktop App

[![Tests](https://github.com/jakob-hendrix/PathfinderWayfinder/actions/workflows/dotnet.yml/badge.svg)](https://github.com/jakob-hendrix/PathfinderWayfinder/actions/workflows/dotnet.yml)

Wayfinder is a desktop application built with .NET MAUI designed to streamline character creation and management for the Pathfinder Roleplaying Game (1st Edition).

Whether you're a veteran player managing a high-level Wizard or a newcomer trying to calculate your first Power Attack penalty, Wayfinder does the heavy lifting so you can focus on the roleplay.

## ✨ Features

- Automated Calculations: Automatically handles modifiers for Ability Scores, Skills, and Saving Throws.
- Feat Tracking: A searchable database (or planned integration) to track prerequisites and active bonuses.
- Inventory Management: Track encumbrance and equipment weights in real-time.
